# お薬手帳 記載票プリント

院内で使う、印刷専用のWebアプリです。

## できること

- レセコンから `氏名 / 生年月日 / 連絡先` をまとめて貼り付け
- 患者ごとにその場で追記
- `利用薬局名 / 薬局電話番号 / 薬剤師氏名 / かかりつけ表示` を端末ごとの既定値として保存
- `move CL-E300-LFX / 80 x 115 mm` を初期プリセットとして用意
- A6相当を初期値にして、レスプリ向けに `幅 / 高さ / 余白` をmmで調整
- ブラウザ印刷でそのまま出力

## 保存方針

- 患者情報はブラウザ保存しません
- 薬局の既定値だけ `localStorage` に保存します

## 起動

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

## 運用メモ

- このPCの `move CL-E300-LFX` では、利用可能な用紙候補に `80 x 115 mm` があるため、初期値をそれに合わせています
- 印刷時はブラウザの余白を最小にしてください
- レスプリ側の用紙設定は、画面の `印刷幅 mm / 印刷高さ mm` と同じ値に合わせてください
- ブラウザだけでは印刷先プリンターの自動選択はできないため、印刷ダイアログで `move CL-E300-LFX` を選んでください
- 患者保存や認証は入れていません
